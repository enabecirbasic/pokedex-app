# Pokedex application
Pokedex React app

> **Live Demo: [https://ena-pokedex.herokuapp.com/](https://ena-pokedex.herokuapp.com/)**

## Getting started

```sh
git clone https://github.com/enabecirbasic/pokedex-app.git
cd pokedex-app
npm install
npm start
```
